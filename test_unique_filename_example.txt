Пример уникальных имен файлов отчетов:

Старый формат:
- Отчет_по_оценке_объекта_Türkiye-Antalya-Alanya-Mahmutlar_20250103_153025.html

Новый уникальный формат:
- Report_123456789_20250103_153025_45721_Turkiye-Antalya-Alanya-Mahmutl.html
  ↳ Report_{telegram_id}_{timestamp}_{unique_id}_{safe_location}.html

Структура нового имени файла:
┌─────────────────────────────────────────────────────────────┐
│ Report_123456789_20250103_153025_45721_Turkiye-Antalya.html │
└─────────────────────────────────────────────────────────────┘
    │        │          │           │              │
    │        │          │           │              └─ Локация (макс 30 символов)
    │        │          │           └─ Уникальный ID (5 цифр)
    │        │          └─ Timestamp (YYYYMMDD_HHMMSS)
    │        └─ Telegram ID пользователя
    └─ Префикс "Report"

Преимущества:
✅ Гарантированная уникальность
✅ Привязка к пользователю
✅ Временная метка
✅ Случайный компонент для избежания коллизий
✅ Безопасные символы для файловой системы
